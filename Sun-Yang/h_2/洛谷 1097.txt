题目描述
某次科研调查时得到了n个自然数，每个数均不超过(1.5×10的9次方 )。

已知不相同的数不超过10000个，现在需要统计这些自然数各自出现的次数，并按照自然数从小到大的顺序输出统计结果。

输入格式
共n+1行。

第一行是整数n，表示自然数的个数；

第2至n+1每行一个自然数。

输出格式
共m行（m为n个自然数中不相同数的个数），按照自然数从小到大的顺序输出。

每行输出2个整数，分别是自然数和该数出现的次数，其间用一个空格隔开。

题目分析
	输入统计数字，但数组大小不够，只能用数组存数字 然后查询每个数字出现多少次


#include<bits/stdc++.h>

using namespace std;
int a[200005];

int main()
{
	int n;
	cin >> n;
	for(int i = 0;i < n;i ++)
	{
		cin >> a[i];
	}
	
	
	sort(a,a+n);
	
	int k = 1;
	
	for(int i = 1 ;i < n;i ++)
	{
		if(a[i]==a[i-1]) k++;
		else if(a[i]!=a[i-1])
		{
			cout << a[i-1] <<" "<< k<<'\n';
			k = 1;
		}
	}
	
	cout <<a[n-1] <<" "<< k;
}
